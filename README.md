# Space Shooter (C Project)

## Description

This project is a simple console-based Space Shooter game developed in C.
The player controls a spaceship and shoots incoming enemies while trying to survive. The game ends when the player loses all lives.

---

## Objectives

- To understand and implement basic concepts of C programming
- To use structures and arrays for managing game entities
- To develop logic for collision detection and scoring
- To implement a simple interactive application

---

## Features

- Player movement (left and right)
- Shooting mechanism
- Enemy spawning and movement
- Collision detection
- Score tracking
- Lives system and game over condition
- High score storage using file handling

---

## Concepts Used

- Structures (`struct`)
- Arrays
- Functions (modular programming)
- Conditional statements and loops
- File handling

---

## How to Run

1. Compile the program:

```bash
gcc space_shooter.c -o game
```

2. Run the executable:

```bash
./game
```

---

## Controls

- `A` → Move Left
- `D` → Move Right
- `Space` → Shoot
- `Q` → Quit

---

## Output

The game is displayed in the console using characters:

- `^` represents the player
- `V` represents enemies
- `|` represents bullets

---

## Conclusion

The Space Shooter game successfully demonstrates the application of C programming concepts in building an interactive program. It helped in understanding structured programming, logic building, and basic game mechanics.
